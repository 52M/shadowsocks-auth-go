shadowsocks-auth-go
===================

A auth front for shadowsocks,which support rfc1929 username/password authentication method


client <-----> shadowsocks-auto-go  <----->  shadowsocks(local)  <----> shadowsocks(server)  <----> Internet


Keywords:

shadowsocks
SOCKS5


Usage
=======

shadowsocks-auto-go -c auth.json
